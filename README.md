# E-commerce Pipeline

End-to-end data pipeline analyzing e-commerce orders. Built as portfolio project for Data Engineer track.

## Tech Stack
- Python 3.11
- pandas
- SQL (PostgreSQL)

## Project Structure
- `data/` — Raw, processed, and output data
- `src/` — Python source code
- `tests/` — Unit tests
- `logs/` — Pipeline logs

## Setup
Install dependencies and run pipeline:

    pip install -r requirements.txt
    python src/pipeline.py
